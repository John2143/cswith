Windows and mabye mac
1. Install [node](https://nodejs.org/)
2. Change the config settings in config.js
3. Click start.bat (should not need admin privelages)
Not Windows
1. sudo apt-get install nodejs
2. vi config.js
3. chmod +x start.sh
4. ./start.sh
